# 🏠 Hostel Mess Management System

A modern web-based application developed to automate and simplify hostel and mess management operations. The system provides a centralized platform for managing room allotments, attendance, mess menu voting, night-out permissions, and complaint handling, reducing paperwork and improving communication between students and hostel authorities.

---

## 📌 Overview

The **Hostel Mess Management System** is designed to digitize daily hostel activities by providing separate dashboards for **Students** and **Administrators**. Students can mark attendance, vote for mess menus, request night-outs, and submit complaints, while administrators can manage rooms, approve requests, monitor attendance, and resolve complaints efficiently.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/SanketBhapkar/Hostel_Management.git
```

---

## Backend Setup

```bash
cd backend
```

* Open the project in **VS Code**, **IntelliJ IDEA**, or **Eclipse**
* Configure the MySQL database
* Update the `application.properties` file
* Run the Spring Boot application

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```
